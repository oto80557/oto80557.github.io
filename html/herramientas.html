Herramientas de software para el docente


Producción de material gráfico:



Inkscape
Esta es una aplicación multiplataforma open source de creación y edición de gráficos vectoriales. Sus características son de orden profesional, aunque de fácil uso, por lo que puede ser utilizado tanto para fines artísticos como para fines técnicos. Por ser gráficos vectoriales, permite crear imágenes de gran calidad y escalables sin pérdida, por lo que es ideal para materiales didácticos gráficos online y offline, aparte de diseñar, maquetar y exportar documentos evaluativos. Es compatible con una gran cantidad de formatos, aunque su formato nativo es una versión propia de SVG.
Fuente: Inkscape, 2020.

GIMP
Esta aplicación (que responde al acrónimo GNU Image Manipulation Program, en español Programa de Manipulación de Imagen GNU), es un editor de fotografías e imágenes open source de índole profesional. Es altamente personalizable, por lo que se puede reprogramar de forma personal o pueden agregársele complementos de terceros. Su principal función dentro del proceso de enseñanza-aprendizaje es el de generar recursos gráficos (online y offline) como material didáctico.
Fuente:GIMP, 2020.

- Tutorial de la herramienta



Medibang Paint Pro
Esta es una aplicación gratuita de dibujo y boceto digital, con énfasis en la elaboración de cómics (y por tanto materiales didácticos de tipo gráfico). Actualmente sólo se encuentra disponible para Windows y Mac OSX. Sus características son de tipo profesional, aunque es fácil de utilizar. Por lo tanto, incluye múltiples brochas, el uso de capas y guías, fondos preestablecidos, entre muchos otras. Además, su procesamiento ligero le permite instalarse y utilizarse en la mayoría de computadoras. Su principal función dentro del proceso de enseñanza-aprendizaje es el de generar recursos gráficos (online y offline) como material didáctico.
Fuente:Medibang, 2020.

Producción de audio y video:




Audacity
Esta es una aplicación open source para la grabación y edición de audio multipista multiplataforma. Este se encuentra en la mayoría de los principales idiomas existentes, y permite el uso de gran variedad de códecs, formatos, y mezcla de audio, además de la inclusión de recursos adicionales propios o de terceros. Su principal función dentro del proceso de enseñanza-aprendizaje es el de generar recursos sonoros (online y offline) como material didáctico.
Fuente:Audacity, 2020.

- Conocer más...
- Tutorial de la herramienta


Kdenlive
Esta aplicación (que responde al acrónimo de KDE Non-Linear Editor, en español Editor no-lineal de KDE) es una aplicación multiplataforma (originaria de GNU/Linux) open source destinada a la edición y montaje de material audiovisual. Esta permite la edición de acceso aleatorio a través del uso de capas, y es compatible con la mayoría de formatos y códecs de audio y video. Posee además la capacidad de convertir videos a otros formatos, según sea la necesidad de la calidad/tamaño del archivo resultante, así como se le pueden adicionar recursos en línea desde el Kdenlive Add-on Installer. Su principal función dentro del proceso de enseñanza-aprendizaje es el de generar recursos sonoros (online y offline) como material didáctico. Los archivos resultantes pueden incluso subirse a los campus virtuales o a youtube.
Fuente: Kdenlive, 2020.

Juegos en el sitio:

Hot Potatoes
Esta es una suite de aplicaciones lúdicas gratuita, que permite crear pruebas educativas digitales de selección múltiple, respuesta corta, pareos, ordenamientos, relleno de espacios y oraciones mezcladas, así como juegos de tipo crucigrama. Permite la exportación de objetos SCORM para plataformas educativas, salvo la versión de Java. Para efectos del uso en el aula virtual, esta herramienta permite generar, por tanto, varios instrumentos de evaluación (online y offline) con formatos lúdicos.
Fuente: Hot Potatoes, 2020.

Editor de texto para sitio Web:

Visual Studio Code
Es una aplicación gratuita de edición de código simplificado. Permite hacer aplicaciones simples y desarrollo web más sencillo, por lo que su principal cualidad es su ciclo rápido de codificación-compilación-depuración. Es ideal para quienes desean generar aplicaciones o páginas web pero no son programadores expertos. Su principal función dentro del proceso de enseñanza-aprendizaje es el de generar recursos interactivos, así como el de poder adaptar páginas web educativas. Aún así, se pueden hacer instrumentos de evaluación interactivos.
Fuente: Microsoft, 2020.

Almacenamiento del proyecto:

Github
Es una plataforma de desarrollo en la nube, con modalidades open source y pagas. Permite administrar proyectos informáticos (así como páginas web), y permite el trabajo colaborativo de forma sincrónica y organizada. Además, permite la adquisición de componentes adicionales para facilitar la codificación a través del Github Marketplace. Su principal función, para efectos del proceso de enseñanza-aprendizaje, es la de almacenar y divulgar ciertos recursos a través de la nube, especialmente las páginas web educativas. Aún así, puede usarse también para generar contenido didáctico en forma de aplicaciones o servicios, así como instrumentos de evaluación interactivos (aunque este es de mayor complejidad).
Fuente: Github, 2020.

Realidad Aumentada:



ARToolkitX
Este proyecto es una colección de herramientas de software que ayudan a resolver problemas y casos fundamentales de la realidad aumentada, incluyendo la capacidad de alinear las coordenadas del espacio virtual con el espacio real (registro geométrico), y calzar la apariencia de los objetos creados en el entorno virtual con el entorno real (registro fotométrico). Está diseñado tanto para programadores expertos como para usuarios nuevos sin experiencia, por lo que es muy intuitivo y fácil de usar. Con esta herramienta se pueden generar recursos interactivos dentro del proceso de enseñanza-aprendizaje, que permitan a los estudiantes aprender utilizando más de un sentido, y de una forma novedosa y creativa.
Fuente: ARToolkitX, 2020.

ARCore
Esta es la plataforma gratuita de Google para el diseño y construcción de experiencias basadas en la realidad aumentada. Se centra fundamentalmente en dispositivos Android (que son los que más utilizan los estudiantes), por lo que le permite a estos percibir e interactuar con el entorno. En algunos casos tiene compatibilidad parcial con iOS. Es más técnico que el ARToolkitX, pero de igual forma es intuitivo y utilizable para la generación de materiales didácticos. Se basa en el uso de la cámara del teléfono, a partir del rastreo de movimiento, el entendimiento del entorno y estimación lumínica. Con esta herramienta se pueden generar recursos interactivos dentro del proceso de enseñanza-aprendizaje, que permitan a los estudiantes aprender utilizando más de un sentido, y de una forma novedosa y creativa.
Fuente: Google, 2020.

Unity
Unity es una plataforma de desarrollo para principiantes y expertos que permite generar experiencias interactivas e inmersivas, desde diferentes formatos, con diferentes propósitos y técnicas. Puede utilizarse sin necesidad de saber programar, pero también puede utilizarse a través del lenguaje programado C#. Esta aplicación, en términos del proceso enseñanza-aprendizaje, permite crear recursos didácticos interactivos, como es el caso de simulaciones y videojuegos educativos. Es el principal soporte a utilizar, de forma gratuita, para poder generar contenidos de realidad aumentada con mayor alcance, al incluir los productos obtenidos desde ARCore y ARToolkitX en aplicaciones más completas y robustas. Además, por su cualidad programable y adaptable, puede utilizarse también para generar nuevos tipos de instrumentos de evaluación.
Fuente: Unity, 2020.
